# 17 — پرسش‌های متداول و پشتیبانی نسخه ۱٫۰

## این برنامه چه کاری انجام می‌دهد؟

موجودی و برنامهٔ مصرفی را که خودتان وارد می‌کنید نگهداری می‌کند و زمان تقریبی پایان موجودی را محاسبه می‌کند. برنامه تشخیص، نسخه‌نویسی یا پیشنهاد تغییر مقدار و زمان مصرف انجام نمی‌دهد.

## چرا تاریخ پایان دقیق و قطعی نیست؟

محاسبه براساس موجودی و برنامهٔ ثبت‌شده است. مصرف واقعی، تغییر دستور پزشک، فراموشی مصرف، قرص آسیب‌دیده یا اطلاعات اشتباه می‌تواند نتیجه را تغییر دهد. در صورت تفاوت، موجودی را دوباره بشمارید و یک «اصلاح موجودی» تازه ثبت کنید.

## تاریخ‌ها چرا شمسی‌اند؟

نمایش و انتخاب تاریخ با تقویم جلالی و ارقام فارسی انجام می‌شود. داخل برنامه زمان‌ها به‌صورت استاندارد `DateTime` ذخیره می‌شوند تا محاسبه، مرتب‌سازی، دیتابیس و اعلان‌ها پایدار بمانند.

## می‌توانم خرید یا شمارش قبلی را ثبت کنم؟

بله. در خرید مجدد یا اصلاح موجودی، تاریخ و ساعت اثر را انتخاب کنید. تاریخ آینده و تاریخی قبل از آخرین مبنای موجودی پذیرفته نمی‌شود؛ برای حفظ تاریخچه، رویدادهای بعدی نباید گذشته را بازنویسی کنند.

## چرا اعلان نمایش داده نمی‌شود؟

- مجوز اعلان باید از داخل برنامه و با انتخاب خودتان فعال شود.
- تنظیمات اعلان Android، Battery Saver و محدودیت‌های سازندهٔ گوشی را بررسی کنید.
- داروی آرشیوشده یا حذف‌شده اعلان فعال ندارد.
- اعلان موجودی دقیقِ پزشکی یا ساعت حیاتی نیست و با زمان‌بندی inexact اجرا می‌شود.

ردکردن مجوز اعلان مانع استفاده از مدیریت دارو نمی‌شود.

## چرا فایل Debug قبلی بزرگ و کند بود؟

Debug شامل ابزارهای hot reload، Dart VM و payloadهای تشخیصی است و معیار نسخهٔ نهایی نیست. برای تست واقعی سرعت و حجم از APK arm64 Release استفاده کنید.

## خطای ناسازگاری امضا هنگام نصب چیست؟

Android فقط زمانی یک APK را به‌عنوان update نصب می‌کند که Application ID و امضای آن با نسخهٔ نصب‌شده سازگار باشد و versionCode افزایش یافته باشد. نسخهٔ Debug یا فایل امضاشده با کلید موقت ممکن است نیاز به uninstall داشته باشد؛ uninstall اطلاعات محلی را حذف می‌کند.

## اطلاعات کجا نگهداری می‌شوند؟

نسخهٔ ۱٫۰ اطلاعات دارویی را داخل پایگاه دادهٔ محلی برنامه روی همان دستگاه نگهداری می‌کند. حساب کاربری، cloud sync، analytics، SDK تبلیغاتی فعال و انتقال اطلاعات دارویی به سرور وجود ندارد.

## چگونه اطلاعات را حذف کنم؟

- برای حذف یک دارو، ابتدا آن را آرشیو و سپس حذف دائمی کنید.
- برای حذف همهٔ اطلاعات دارویی، وارد «حریم خصوصی و اطلاعات برنامه» شوید و فرمان حذف دائمی را پس از خواندن تأیید اجرا کنید.
- پاک‌کردن data برنامه یا uninstall نیز دادهٔ محلی را حذف می‌کند.

حذف داخلی دربارهٔ داده‌های Android، فروشگاه یا سرویس ثالث خارج از برنامه ادعایی ندارد.

## آیا backup ابری وجود دارد؟

خیر. Auto Backup و انتقال خودکار دادهٔ برنامه غیرفعال شده‌اند. پیش از uninstall یا تعویض امضای تست، فرض کنید دادهٔ محلی قابل‌بازیابی نیست. Export/restore کاربرمحور در Issue `#45` طراحی می‌شود و جزو نسخهٔ ۱٫۰ نیست.

## متن بزرگ یا TalkBack مشکل دارد؛ چه گزارشی بدهم؟

بدون ارسال اطلاعات دارویی واقعی، این موارد را بنویسید:

- مدل گوشی و نسخهٔ Android؛
- نسخهٔ برنامه؛
- اندازهٔ متن و Display size؛
- روشن/خاموش‌بودن TalkBack؛
- نام صفحه و مراحل دقیق؛
- تصویر یا ویدئوی بازتولید با دادهٔ ساختگی.

## برای خطای فنی چه اطلاعاتی نفرستم؟

نام داروی واقعی، تشخیص، دوز، برنامهٔ مصرف، موجودی، یادداشت، تصویر نسخه، اعلان واقعی، keystore، رمز، GitHub Secret یا اطلاعات هویتی را در issue عمومی قرار ندهید.

## در وضعیت فوری پزشکی چه کنم؟

این برنامه سرویس اورژانس یا مشاورهٔ پزشکی نیست. برای تصمیم دربارهٔ مصرف، قطع یا تغییر دارو با پزشک یا داروساز تماس بگیرید و در وضعیت اضطراری از خدمات فوریت‌های پزشکی محل زندگی استفاده کنید.
