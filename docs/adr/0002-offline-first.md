# ADR-0002 — معماری Offline-first بدون حساب کاربری

- Status: Accepted
- Date: 2026-07-18

## Context

ارزش اصلی محصول به اینترنت نیاز ندارد و داده‌ها ماهیت سلامت شخصی دارند. backend در MVP هزینه، پیچیدگی و ریسک حریم خصوصی ایجاد می‌کند.

## Decision

دیتابیس محلی منبع حقیقت است و MVP حساب کاربری یا backend ندارد.

## Consequences

- اپ حتی بدون اینترنت کامل کار می‌کند.
- هزینه زیرساخت نزدیک به صفر است.
- حذف اپ یا خرابی دستگاه می‌تواند موجب از دست‌رفتن داده شود؛ export/backup در مراحل بعد اضافه می‌شود.
- تبلیغات هنگام قطع اینترنت نمایش داده نمی‌شوند و این شکست نباید UX را مختل کند.
