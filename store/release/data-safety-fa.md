# پیش‌نویس افشای داده و Data Safety نسخه ۱٫۰

## رفتار واقعی نسخه کاندید

- اطلاعات دارویی در SQLite محلی برنامه ذخیره می‌شوند.
- حساب کاربری، همگام‌سازی ابری یا سرور دارویی وجود ندارد.
- SDK تبلیغاتی یا analytics فعال نیست.
- مجوز اینترنت در نسخه کاندید مجاز نیست و CI حضور آن را رد می‌کند.
- اعلان‌ها اختیاری‌اند و رد مجوز مانع قابلیت‌های اصلی نمی‌شود.
- backup و device transfer برای داده برنامه غیرفعال‌اند تا طراحی رمزگذاری‌شده و رضایت‌محور مستقلی ایجاد شود.

## داده‌های واردشده توسط کاربر

- نام دارو
- واحد و موجودی
- برنامه مصرف واردشده
- توضیحات اختیاری
- تاریخچه خرید مجدد، اصلاح و تغییر برنامه

این داده‌ها برای عملکرد محلی برنامه استفاده می‌شوند و در نسخه کاندید از دستگاه خارج نمی‌شوند.

## پاسخ پیشنهادی فرم فروشگاه

پاسخ نهایی باید بر اساس APK/AAB همان نسخه و متن دقیق فرم فروشگاه ثبت شود. پیش از انتشار، ناشر باید permission inventory، dependency inventory و سیاست حریم خصوصی HTTPS را با فرم فروشگاه تطبیق دهد. اضافه‌شدن تبلیغات یا analytics این پاسخ‌ها را تغییر می‌دهد و به بازبینی جدید نیاز دارد.
